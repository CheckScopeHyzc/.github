# CheckScopeHyzc

面向 C/C++ 工程代码的静态分析与可追溯智能审查组织，核心项目为 `SourceTrace` 与 `CheckScopeAI`。

## 我们在做什么

- 构建源码级调用关系与可追溯能力
- 融合规则体系与大模型能力进行智能审查
- 产出结构化、可复现的风险分析结果

## 核心项目

- [SourceTrace](https://github.com/CheckScopeHyzc/SourceTrace)：函数调用链、调用树与源码溯源能力。
- [CheckScopeAI](https://github.com/CheckScopeHyzc/CheckScopeAI)：面向工程质量场景的规则+LLM 检测与报告框架。

## 目标

- 提升问题发现效率
- 降低人工审查成本
- 支持团队化持续集成落地
